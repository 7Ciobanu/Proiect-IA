# Connect 4 – Minimax cu Alpha-Beta

## Descriere
Acest proiect implementează jocul **Connect 4**, în care un jucător joacă împotriva calculatorului.

Calculatorul folosește **algoritmul Minimax cu retezare Alpha-Beta** pentru a alege mutările optime. Utilizatorul poate seta **numărul de niveluri de căutare**, influențând dificultatea jocului.
